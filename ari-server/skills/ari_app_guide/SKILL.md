# 📱 앱 사용법 (App Guide)
이 스킬은 사용자가 ARI 앱의 전체 사용법, 첫 설정 방법, 각 탭의 역할, 앱/스킬 관리 방법을 물어볼 때 참고하는 문서형 가이드입니다.

## 언제 이 스킬을 참고하나요?

- 사용자가 "아리앱 사용법 알려줘", "이 앱에서 뭐 할 수 있어?", "처음 실행하면 뭘 해야 해?"처럼 전체 사용법을 묻는 경우
- 사용자가 특정 화면이나 탭의 용도를 묻는 경우
- 사용자가 앱, 스킬, 아바타, 서버 설정 위치를 찾지 못하는 경우

## ARI 앱 한눈에 보기

ARI 앱은 로컬에서 동작하는 AI 비서 데스크톱 앱입니다. 메인 화면은 크게 다음 4개 탭으로 구성됩니다.

1. `Chat`
2. `Place`
3. `Avatar`
4. `Settings`

기본 사용 흐름은 다음과 같습니다.

1. 앱 실행
2. 로컬 에이전트가 켜져 있는지 확인
3. AI 모델/API 연결 설정
4. `Chat` 탭에서 대화 시작
5. 필요하면 `Avatar`, `Apps`, `Skills`, `Settings`로 세부 관리

## 처음 실행했을 때

### 1. 앱 실행

- ARI 앱을 실행하면 로컬 서버(`ari-server`)가 함께 준비됩니다.
- 창을 닫아도 완전히 종료되지 않고 숨김 처리될 수 있습니다.
- 시스템 트레이에서 다시 열거나 숨길 수 있습니다.

### 2. 에이전트가 꺼져 있으면

- 서버가 꺼져 있으면 `Chat` 같은 일부 기능은 바로 사용할 수 없습니다.
- 이 경우 `Settings`로 이동해 로컬 에이전트 상태를 확인하고 제어합니다.

### 3. AI 모델/API 연결

- API 키나 OAuth 연결이 없으면 채팅 전에 모델 설정이 필요합니다.
- `Settings > Model Settings`에서 사용할 AI provider와 모델을 설정합니다.
- provider는 여러 개를 등록할 수 있고, 우선순위 순서대로 정렬할 수 있습니다.

## Chat 탭 사용법

`Chat`은 ARI와 직접 대화하는 기본 화면입니다.

- 질문이나 요청을 입력하면 현재 선택된 아바타 기준으로 응답합니다.
- 서버가 꺼져 있으면 입력창 대신 서버 시작 안내가 보일 수 있습니다.
- 모델 설정이 끝나지 않았으면 설정 유도 화면이 먼저 표시됩니다.
- 초기 추천 문구를 눌러 바로 대화를 시작할 수 있습니다.

### Chat 탭에서 할 수 있는 대표 작업

- 일반 질문 답변
- 파일 찾기, 시스템 작업 요청
- 알림/예약 요청
- 앱 설치 요청
- 연결된 앱 제어 요청

### 예시 요청

- "사용법을 알려줘"
- "1분 뒤 알려줘"
- "디스크 용량 확인해줘"
- "비트코인 거래앱 설치해줘"

## Place 탭 사용법

`Place`는 ARI와 연결된 요소들을 공간처럼 보여주는 화면입니다.

- 아바타들이 방 안에 배치되어 보입니다.
- Home Assistant 기기가 연동되어 있으면 기기 아이콘도 함께 표시됩니다.
- 상단 바에서 설치된 앱 목록을 볼 수 있습니다.
- 우하단 새로고침 버튼으로 아바타/기기 배치를 다시 섞을 수 있습니다.
- 좌하단 Home Assistant 버튼으로 스마트홈 연동을 시작할 수 있습니다.

### Home Assistant 연동

연동 시 다음 정보가 필요합니다.

- Home Assistant URL
- Long-Lived Access Token

연동이 완료되면 Place 화면에서 일부 기기를 눌러 제어할 수 있습니다.

## Avatar 탭 사용법

`Avatar`는 현재 에이전트와 관련된 정보를 관리하는 화면입니다. 내부에 여러 하위 탭이 있습니다.

### Profile

- 현재 아바타의 이름과 프로필 성격(persona) 같은 기본 정보를 봅니다.

### Logs

- 현재 아바타와 관련된 기록과 로그를 확인합니다.

### Memory

- 장기 기억이나 메모리 상태를 확인합니다.

### Schedule

- 예약된 작업이나 스케줄 관련 내용을 확인합니다.

### Apps

- 설치된 앱 목록을 확인합니다.
- 앱이 현재 연결되어 있는지(`연결됨`/`오프라인`) 상태를 볼 수 있습니다.
- 오프라인 앱은 실행 버튼으로 켤 수 있습니다.
- 설치된 앱이 없다면 보통 `~/.ari-agent/apps` 폴더를 확인하면 됩니다.

### Skills

- 등록된 스킬 목록을 확인합니다.
- 각 스킬 설명과 사용 도구 수를 확인할 수 있습니다.
- 커스텀 스킬은 삭제할 수 있습니다.
- 등록된 스킬이 없다면 에이전트의 `skills` 폴더를 확인하면 됩니다.

### Tools

- 현재 등록된 도구 목록과 설명을 확인합니다.

### Channels

- 채널 관련 연결 상태나 구성을 확인하는 영역입니다.

### Settings

- 현재 아바타 기준의 데이터 관리, 초기화, 삭제 작업을 수행합니다.

현재 아바타 화면 하단에서는 다음도 할 수 있습니다.

- 현재 선택된 에이전트 확인
- `목록` 버튼으로 다른 아바타 선택

## Avatar > Settings 상세

이 화면은 현재 아바타의 데이터 관리용입니다.

- `기억 초기화`: 장기 기억 벡터 데이터를 초기화
- `대화 초기화`: 현재 아바타와의 채팅 로그 삭제
- `아바타 삭제`: 현재 아바타 전체 삭제
- `설정 폴더 열기`: `.ari-agent` 폴더 열기

주의:

- 기본 아바타 `ARI`는 삭제할 수 없습니다.
- 기억/대화 초기화는 되돌릴 수 없습니다.

## Settings 탭 사용법

메인 `Settings`는 앱 전체 설정 화면입니다.

### 1. Model Settings

- AI provider 추가
- API 키 또는 OAuth 방식 설정
- 모델 선택
- provider 우선순위 변경
- 설정 저장

### 2. Agent Settings

- 로컬 에이전트 상태 확인
- 서버 포트 변경
- 실시간 에이전트 로그 확인
- 로그 복사

포트를 바꾸면 보통 에이전트 재시작이 필요합니다.

### 3. Appearance

- 앱의 화면 스타일
- 윈도우 핀 고정 같은 표시 관련 설정

### 4. Notification

- 데몬 상태에서 받는 OS 알림 설정

### 5. App Intro

- ARI Agent 소개와 버전 정보 확인

## 앱/스킬 설치와 관리

### 앱

- 설치된 앱은 보통 `~/.ari-agent/apps` 아래에서 관리됩니다.
- 앱이 설치되어 있어도 실행 전에는 `오프라인`으로 보일 수 있습니다.
- 앱 실행 후 서버와 연결되면 `연결됨` 상태로 바뀔 수 있습니다.

### 스킬

- 기본 스킬과 커스텀 스킬이 함께 표시될 수 있습니다.
- 커스텀 스킬은 목록에서 삭제 가능합니다.
- 새 스킬을 추가한 뒤에는 서버가 다시 읽어야 목록에 반영될 수 있습니다.

## 자주 묻는 사용 흐름

### "그냥 대화만 하고 싶어요"

1. 앱 실행
2. `Settings > Model Settings`에서 AI 연결
3. `Chat` 탭에서 질문 입력

### "외부 앱을 같이 쓰고 싶어요"

1. 앱 설치
2. `Avatar > Apps`에서 설치 여부 확인
3. 필요하면 실행 버튼으로 앱 실행
4. `Chat`에서 해당 앱을 써 달라고 요청

### "스마트홈을 연결하고 싶어요"

1. `Place` 탭 열기
2. Home Assistant 버튼 누르기
3. URL과 토큰 입력
4. 연동 후 기기 상태 확인 및 제어

### "아바타별로 관리하고 싶어요"

1. `Avatar` 탭 열기
2. 하단 `목록`에서 아바타 선택
3. 각 아바타의 로그, 메모리, 설정을 분리해서 관리

## 데이터 위치

사용 중 자주 보게 되는 로컬 폴더는 다음과 같습니다.

- `~/.ari-agent`
- `~/.ari-agent/apps`
- `~/.ari-agent/skills`

## 답변 방식 가이드

사용자가 전체 사용법을 물으면 위 내용을 탭 기준으로 짧게 정리해서 안내합니다.
특정 기능만 물으면 해당 섹션만 골라 간단히 설명합니다.
