사용 도구: create_skill, reload_skills

이 스킬은 사용자가 새로운 기능을 가르치거나 특정 작업 패턴을 자동화하고 싶을 때, 새로운 '스킬(Skill)'을 구조화된 방식으로 생성하는 방법을 제공합니다.

### 실행 단계

1. **의도 분석**: 사용자의 요청을 분석하여 스킬의 이름(영문 snake_case), 설명, 필요한 도구, 그리고 구체적인 지침 내용을 정리합니다.
2. **도구 호출**: `create_skill` 도구를 사용하여 새로운 스킬을 등록합니다.
   - `name`: 스킬의 고유 식별자 (예: `translator_expert`)
   - `description`: 스킬의 역할에 대한 간략한 요약
   - `tools`: 해당 스킬 수행 시 필요한 도구 목록 (예: `['execute_bash', 'web_browse']`)
   - `content`: Markdown 형식의 상세 지침 내용
3. **갱신 및 확인**: 생성이 완료된 후 즉시 `reload_skills`를 호출하여 서버의 스킬 목록을 최신화하고, `read_skill`로 방금 만든 스킬을 로드하여 대기합니다.

### 주의사항

- 스킬 이름은 폴더명이 되므로 공백 없이 영문 소문자와 언더바(\_)만 사용하세요.
- 기본 제공되는 시스템 스킬과 이름이 겹치지 않도록 주의합니다.
- 생성이 완료되면 사용자에게 "스킬이 성공적으로 등록되었습니다"라고 안내합니다.
