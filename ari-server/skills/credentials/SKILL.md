# Credentials Management Skill

이 스킬은 사용자의 아이디와 비밀번호와 같은 민감한 정보를 macOS 시스템 키체인에 안전하게 관리하는 기능을 제공합니다.

사용 도구: set_credential, get_credential, delete_credential

## 가이드라인

1. 사용자가 특정 사이트(예: 페이스북, 인스타그램 등)에 로그인이 필요한 작업을 요청하면, 먼저 `get_credential`을 통해 저장된 정보가 있는지 확인하세요.
2. 정보가 없다면 사용자에게 아이디와 비밀번호를 물어본 뒤 `set_credential`로 저장할지 제안하세요.
3. 보안을 위해 비밀번호 자체를 채팅창에 노출하기보다는 "비밀번호를 성공적으로 가져왔습니다"와 같이 안내하고, 실제 자동화 도구에만 전달하세요.
4. 자격 증명을 업데이트해야 할 때는 `set_credential`을 다시 호출하면 기존 항목이 덮어씌워집니다.
