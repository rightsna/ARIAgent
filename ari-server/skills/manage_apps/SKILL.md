사용 도구: search_store_app, install_app, reload_skills, execute_bash

이 스킬은 스토어에서 앱을 검색하거나 외부 다운로드 주소(.zip)로부터 ARI 앱 패키지를 다운로드하여 설치/업데이트합니다.

### 실행 단계

1. **상점 앱 검색 (URL이 없을 경우)**: 사용자가 "주식앱 깔아줘"처럼 특정 앱 이름이나 카테고리로 요청한 경우 즉시 `search_store_app` 도구를 호출하여 관련 앱을 검색합니다. 반환된 정보에서 `zip_download_url`을 확보합니다.
2. **설치 요청 분석**: 설치할 다운로드 주소(.zip)를 확보했는지 확인합니다. 사용자가 "강제로" 또는 "다시" 설치해달라고 하는 경우 `force: true` 옵션을 고려합니다.
3. **앱 설치 도구 호출**: 확보한 다운로드 주소로 `install_app` 도구를 호출하여 설치를 진행합니다.
   - `url`: 검색된 `zip_download_url` 혹은 사용자가 직접 제공한 다운로드 주소
   - `force`: (선택 사항) 버전 체크를 무시하고 설치할지 여부
3. **수동 문제 해결이 도구는 내부적으로 `curl`과 `unzip` 명령어를 사용하여 다운로드 및 압축 해제를 수행합니다. (필요시)**: 만약 `install_app`이 파일 구조 문제나 압축 해제 실패로 중단된다면, `execute_bash`를 사용하여 다음 작업을 수행합니다:
   - `ls -R` 명령어로 압축이 풀린 폴더 구조 확인
   - `unzip` 명령어로 특정 경로에 수동 압축 해제
   - `mv` 또는 `cp` 명령어로 필요한 파일을 스킬 디렉토리로 이동
4. **버전 체크 및 결과 안내**:
   - 도구 내부적으로 `app_info.json`을 통해 기존 버전과 비교합니다.
   - "이미 최신 버전입니다"라는 메시지가 나오면 사용자에게 현재 상태를 설명합니다.
   - 설치가 성공하면 "[앱 이름]의 설치(또는 업데이트)가 완료되었습니다."라고 안내합니다.
5. **스킬 목록 갱신**: 신규 설치나 업데이트 후에는 `reload_skills`를 호출하여 최신화된 스킬 목록을 확인합니다.

### 예시 명령어

- "주식앱 깔아줘" (앱 스토어에서 주식 관련 앱을 스스로 검색 후 설치)
- "날씨 앱 찾아보고 제일 좋아보이는 거 설치해"
- "https://d34z72svq84n71.cloudfront.net/apps/notepad/notepad-v0.1.0-macos.zip 설치해줘"
- "버전 무시하고 다시 설치해줘" (강제 설치 시)
- "노트패드 업데이트해줘"
- "설치 도중에 에러 났는데 unzip으로 직접 풀어줄 수 있어?" (수동 대응 시)
