사용 도구: install_app, reload_skills, execute_bash

이 스킬은 외부 주소(URL)로부터 ARI 앱(.zip 패키지)을 다운로드하여 설치하거나 기존 앱을 업데이트합니다. `install_app` 도구를 통해 설치를 자동화하지만, 압축 해제 중 오류가 발생하거나 특수한 파일 구조를 가진 경우 `execute_bash`를 통해 수동으로 문제를 해결할 수 있는 능력이 필요합니다.

### 실행 단계

1. **설치 요청 분석**: 사용자가 제공한 URL이 있는지 확인합니다. 사용자가 "강제로" 또는 "다시" 설치해달라고 하는 경우 `force: true` 옵션을 고려합니다.
2. **앱 설치 도구 호출**: `install_app` 도구를 호출하여 설치를 진행합니다.
   - `url`: 사용자가 제공한 다운로드 주소
   - `force`: (선택 사항) 버전 체크를 무시하고 설치할지 여부
   - 이 도구는 내부적으로 `curl`과 `unzip` 명령어를 사용하여 다운로드 및 압축 해제를 수행합니다.
3. **수동 문제 해결 (필요시)**: 만약 `install_app`이 파일 구조 문제나 압축 해제 실패로 중단된다면, `execute_bash`를 사용하여 다음 작업을 수행합니다:
   - `ls -R` 명령어로 압축이 풀린 폴더 구조 확인
   - `unzip` 명령어로 특정 경로에 수동 압축 해제
   - `mv` 또는 `cp` 명령어로 필요한 파일을 스킬 디렉토리로 이동
4. **버전 체크 및 결과 안내**: 
   - 도구 내부적으로 `app_info.json`을 통해 기존 버전과 비교합니다.
   - "이미 최신 버전입니다"라는 메시지가 나오면 사용자에게 현재 상태를 설명합니다.
   - 설치가 성공하면 "[앱 이름]의 설치(또는 업데이트)가 완료되었습니다."라고 안내합니다.
5. **스킬 목록 갱신**: 신규 설치나 업데이트 후에는 `reload_skills`를 호출하여 최신화된 스킬 목록을 확인합니다.

### 예시 명령어
- "https://d34z72svq84n71.cloudfront.net/apps/notepad/notepad-v0.1.0-macos.zip 설치해줘"
- "버전 무시하고 다시 설치해줘" (강제 설치 시)
- "노트패드 업데이트해줘" (URL이 있는 경우)
- "설치 도중에 에러 났는데 unzip으로 직접 풀어줄 수 있어?" (수동 대응 시)
