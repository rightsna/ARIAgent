# Scheduling Skill

일정 예약 및 알림 설정을 관리하기 위한 스킬입니다. 반복되는 작업(크론탭)이나 일회성 타이머/알림을 등록할 수 있습니다.

사용 도구: `register_schedule`, `register_one_off_schedule`

## 💡 사용 지침

1. **반복 일정 (register_schedule)**: 매일 특정 시간이나 주기적으로 실행해야 하는 작업(예: "매일 아침 9시에 뉴스 브리핑")에 사용합니다.
2. **일회성 알림 (register_one_off_schedule)**: "5분 뒤에 알려줘", "내일 오후 1시에 전화 예약 확인해줘"와 같이 한 번만 실행하고 종료될 작업에 사용합니다.
3. **시간 계산 도움**: `register_one_off_schedule`을 사용할 때, 현재 시간으로부터 목표 시간까지의 차이를 '분(minutes)' 단위로 계산하여 입력해야 합니다. (스스로 계산이 어려우면 사용자에게 현재 시각을 물어보거나 추측하지 마세요.)
4. **확인 응답**: 일정을 등록한 후에는 사용자에게 언제 어떤 내용으로 등록되었는지 명확하게 피드백을 주어야 합니다.
