# sys_monitor

사용 도구: execute_bash

macOS 시스템 상태, 디스크 용량, 메모리 점유율, 네트워크 상태를 실제 명령 실행으로 확인할 때 사용하는 스킬이다.

사용 규칙:
- 시스템 상태 관련 질문에는 추측하지 말고 `execute_bash`로 실제 값을 확인한다.
- 필요한 명령은 `uptime`, `df -h`, `vm_stat`, `top -l 1`, `ps` 등 상황에 맞게 선택한다.
- 명령 결과를 그대로 나열하지 말고 핵심 지표를 추려서 요약한다.
- 이상 징후가 보이면 원인 후보와 다음 확인 포인트를 짧게 덧붙인다.
